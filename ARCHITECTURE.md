# System Architecture: Alternating Layer Freezing

## Overview

This project investigates whether **GAN-style alternating optimization** can improve training in standard feedforward neural networks. In GANs, the generator and discriminator are trained alternately to prevent instability. This project applies a similar principle: **freeze half the network while training the other half**, then switch.

## Training Strategies Compared

### 1. Standard Training
```
Epoch 1: Update [Layer1, Layer2, Layer3, Layer4]
Epoch 2: Update [Layer1, Layer2, Layer3, Layer4]
Epoch 3: Update [Layer1, Layer2, Layer3, Layer4]
...
```
All parameters receive gradients at every step.

### 2. Alternating Training (Epoch-Level)
```
Epoch 1: Update [Layer1, Layer2] â”‚ Freeze [Layer3, Layer4]
Epoch 2: Freeze [Layer1, Layer2] â”‚ Update [Layer3, Layer4]
Epoch 3: Update [Layer1, Layer2] â”‚ Freeze [Layer3, Layer4]
...
```
Switch which half is trained at each epoch.

### 3. Alternating Training (Batch-Level)
```
Batch 1: Update [Layer1, Layer2] â”‚ Freeze [Layer3, Layer4]
Batch 2: Freeze [Layer1, Layer2] â”‚ Update [Layer3, Layer4]
Batch 3: Update [Layer1, Layer2] â”‚ Freeze [Layer3, Layer4]
...
```
Switch which half is trained at each batch (more frequent switching).

## Key Implementation Details

### Layer Freezing Mechanism
```python
def freeze_first_half(self):
    for param in self.fc1.parameters():
        param.requires_grad = False  # Stop gradient computation
    for param in self.bn1.parameters():
        param.requires_grad = False
```

When `requires_grad=False`:
- Forward pass still works normally
- Backward pass skips gradient computation for these parameters
- Optimizer updates are not applied to frozen parameters

### Optimizer Recreation
After changing which parameters are frozen, the optimizer is recreated with only trainable parameters:
```python
trainable_params = [p for p in model.parameters() if p.requires_grad]
optimizer = optim.Adam(trainable_params, lr=learning_rate)
```

## Research Questions & Findings

| Question | Finding |
|----------|---------|
| Does alternating training speed up convergence? | Minimal effect; backward pass still traverses full network |
| Does it improve final accuracy? | Generally comparable or slightly lower than standard training |
| Why does this work for GANs but not here? | GANs have adversarial objectives; standard NNs have cooperative layers |

## Expected Results

The experiments demonstrate that while alternating layer freezing is valuable for adversarial training (GANs), it does not provide significant benefits for standard supervised learning where all layers work cooperatively toward the same objective.
